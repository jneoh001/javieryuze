Each cinema file will be titled: 'cineplex_XXX'.
XXX is the cineplex ID.
An example is 'cineplex_DTE' for Downtown East's cineplex.

In each cinepelx file, please follow the format below to create it in text.

==========
Format
==========
Cineplex Name
Cineplex Address (single line string)
<Each row where row is a cinema ID 'cinema_XXX_YYY'>

==========
Example:
==========
cineplex_CAT <---- FILENAME

The Cathay
Handy Rd, Former Cathay Building, Singapore 229233
CAT_001
CAT_002
CAT_003
